# 🚀 TaskFlow Pro – Advanced Backend Project Plan

(Express + TypeScript + PostgreSQL | Clean Architecture | Raw SQL)

---

## 🧭 1. Project Overview

TaskFlow Pro একটি এন্টারপ্রাইজ-গ্রেড প্রজেক্ট ম্যানেজমেন্ট সিস্টেম যেখানে ইউজাররা টিমে কাজ করতে পারবে, টাস্ক ট্র্যাক করতে পারবে এবং রিয়েল-টাইম অ্যাক্টিভিটি মনিটর করতে পারবে।

এই সিস্টেমে থাকবে:

* 👤 User Dashboard
* 🧑‍💼 Admin Dashboard
* 🧑‍💻 Manager Dashboard
* 📊 Performance & Analytics System
* 💬 Feedback System

---

## 🏗️ 2. Tech Stack

* Runtime: Node.js
* Language: TypeScript
* Framework: Express.js
* Database: PostgreSQL (using pg driver)
* Auth: JWT (Access & Refresh Tokens)
* Security: Bcrypt (Hashing), CORS, Helmet
* Validation: Zod
* Logging: Winston or Morgan

## 👥 4. Role System (Core Concept)

এই সিস্টেমে ৩ ধরনের ইউজার থাকবে:

### 🧑 User (Member)

* নিজের assigned task দেখতে পারবে
* task update করতে পারবে
* feedback দেখতে পারবে
* নিজের performance dashboard দেখতে পারবে

---

### 🧑‍💻 Manager

* নিজের team / project manage করবে
* task assign করতে পারবে
* feedback দিতে পারবে
* team performance দেখতে পারবে
* analytics access থাকবে

---

### 🧑‍💼 Admin

* full system control
* সব project access
* user management (create/block/delete)
* global analytics dashboard
* system performance monitoring

## 🗄️ 6. Advanced Database Design (SQL Only)

### 👤 users

id, name, email, password, role, is_active, last_login, created_at

### 🏢 projects

id, title, description, owner_id, slug, is_deleted, created_at

### 👥 project_members

project_id, user_id, role, joined_at

### 📋 boards

id, name, project_id, order_index

### ✅ tasks

id, title, description, status, priority, board_id, creator_id, assigned_to, due_date, created_at

### 📝 activity_logs

id, action, entity_type, entity_id, user_id, details, created_at

### 📊 user_monthly_performance

id, user_id, month, total_tasks, completed_tasks, failed_tasks, success_rate, productivity_score

### 💬 task_feedbacks

id, task_id, user_id, given_by, rating, comment, created_at
